# 📊 Sales Data Analysis in Excel – By Christine Wambui

## 📁 Project Description
This beginner-friendly project showcases my growing Excel skills in data analysis. It involves analyzing product sales using basic formulas and logical operations.

The dataset contains sales transactions for different products across regions, and includes both raw data and calculated insights.

---

## ✅ Skills Demonstrated
- Calculating total sales using `=QUANTITY * PRICE`
- Using Excel functions like `SUM`, `AVERAGE`, `MAX`, `MIN` to summarize data
- Creating a new column to classify sales as **Big** or **Small** using the `IF` function
- Combining text values (e.g., name and region) using `&`
- Extracting text (e.g., city codes) using `LEFT`
- Organizing clean tabular data

---
